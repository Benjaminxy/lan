# Cheatsheet: i, på och om

---

## 1. Använd `i` för länder, städer och områden

Vi använder **i** när vi pratar om platser med gränser.

| Exempel |
|---------|
| i Sverige |
| i Stockholm |
| i Malmö |
| i Iran |
| i ett område |

---

## 2. Använd `i` när något är inuti ett rum eller en behållare

**i** används när något är fysiskt inne i något.

| Exempel |
|---------|
| i köket |
| i klassrummet |
| i väskan |
| i bilen |
| i rymden |

---

## 3. Använd `på` för ytor

**på** används när något ligger eller sitter ovanpå en yta.

| Exempel |
|---------|
| på bordet |
| på golvet |
| på väggen |
| på stolen |
| på tredje våningen |

---

## 4. Använd `på` för gator, torg och adresser

Gator och torg ses som ytor eller linjer.

| Exempel |
|---------|
| på Storgatan 5 |
| på Drottninggatan |
| på torget |
| på gatan |

---

## 5. Använd `på` för öar och berg

| Exempel |
|---------|
| på Öland |
| på Gotland |
| på Island |
| på Kebnekaise |

---

## 6. Använd `på` för institutioner och arbetsplatser

När platsen ses som en arbetsplats, institution eller plats för ett ärende.

| Exempel |
|---------|
| på sjukhuset |
| på posten |
| på universitetet |
| på biblioteket |
| på jobbet |
| på bion |

---

## 7. Positiv tid = `i`

När något **har pågått** under en tid.

> Jag har bott här **i** fem år.
> Hon har studerat svenska **i** två månader.
> Vi har väntat **i** en timme.

---

## 8. Negativ tid = `på`

När något **inte har hänt** under en viss tid.

> Jag har inte rökt **på** tio år.
> Vi har inte träffats **på** länge.
> Han har inte sovit bra **på** flera dagar.

---

## 9. Prestationstid = `på`

När man säger hur snabbt något blev klart.

> Han sprang milen **på** 40 minuter.
> Jag skrev klart uppgiften **på** två timmar.
> Hon lärde sig låten **på** en dag.

---

## 10. Framtid = `om`

När något ska hända **efter** en viss tid.

> Vi ses **om** tio minuter.
> Provet börjar **om** två dagar.
> Jag fyller år **om** tre månader.

---

## 11. Aktiviteter, möten och lektioner = `på`

| Exempel |
|---------|
| Jag är på fest. |
| Jag är på kalas. |
| Jag är på möte. |
| Jag sitter på lektion. |
| Jag är på kurs. |
| Vi är på föreläsning. |
| Hon är på seminarium. |
| De är på konsert. |

> **OBS!** Undantag: *Jag sitter **i** möte.* — fast uttryck som betyder att man är upptagen.

---

## Extra viktiga regler

### Vanor och återkommande tider = `på`

| Exempel |
|---------|
| på morgonen |
| på kvällen |
| på helgerna |
| på vintern |
| på sommaren |

### Kommande årstid = `i`

| Exempel |
|---------|
| i sommar |
| i höst |
| i vinter |
| i vår |

> **Skillnad:**
> - *På sommaren badar jag ofta.* → varje sommar
> - *I sommar ska jag resa.* → kommande sommar

### Månader = `i`

| Exempel |
|---------|
| i januari |
| i maj |
| i juli |
| i oktober |

### Frekvens

| Tid | Preposition | Exempel |
|-----|-------------|---------|
| Vecka / Månad | **i** | en gång **i** veckan, två gånger **i** månaden |
| År | **om** | en gång **om** året, två gånger **om** året |

### Tryckt media = `i`

Böcker och tidningar ses som behållare för text.

| Exempel |
|---------|
| i boken |
| i tidningen |
| i artikeln |
| i DN |

### Digitala medier = `på`

Internet och digitala plattformar ses som ytor.

| Exempel |
|---------|
| på nätet |
| på TV |
| på Instagram |
| på YouTube |
| på hemsidan |

---

## Snabbregel

| Preposition | Regel | Exempel |
|-------------|-------|---------|
| **i** | inuti eller inom ett område, positiv tid | i Sverige, i köket, i boken, i fem år |
| **på** | yta, plats, institution, aktivitet, prestationstid | på bordet, på sjukhuset, på fest, på två timmar, på morgonen |
| **om** | framtid eller årsfrekvens | om tio minuter, en gång om året |
